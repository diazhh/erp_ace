# ⚖️ Cumplimiento Regulatorio (Compliance)

## ¿Qué es este módulo?

El módulo de **Compliance** permite gestionar el cumplimiento de regulaciones, políticas internas, permisos, certificaciones y auditorías. Es esencial para empresas del sector petrolero que deben cumplir con múltiples normativas gubernamentales y estándares de la industria.

Piense en este módulo como su "centro de cumplimiento": asegura que la empresa cumpla con todas las regulaciones, mantiene los permisos vigentes y documenta las auditorías.

## ¿Para quién es útil?

- **Compliance Officer**: Para gestionar cumplimiento regulatorio
- **Legal**: Para controlar permisos y licencias
- **HSE**: Para cumplimiento ambiental y de seguridad
- **Auditoría**: Para gestionar auditorías internas y externas

## ¿Qué puedo hacer aquí?

### Políticas
- **Crear políticas** internas
- **Controlar versiones**
- **Flujo de aprobación**
- **Distribución y reconocimiento**

### Permisos y Licencias
- **Registrar permisos** gubernamentales
- **Controlar vencimientos**
- **Alertas de renovación**
- **Documentación adjunta**

### Certificaciones
- **Gestionar certificaciones** (ISO, OHSAS, etc.)
- **Controlar vigencia**
- **Auditorías de certificación**

### Auditorías
- **Programar auditorías** internas y externas
- **Registrar hallazgos**
- **Seguimiento de acciones**
- **Reportes de auditoría**

### Reportes Regulatorios
- **Generar reportes** para entes gubernamentales
- **Controlar fechas** de entrega
- **Historial de reportes**

## Conceptos Importantes

### Categorías de Políticas

| Categoría | Descripción |
|-----------|-------------|
| **HSE** | Salud, Seguridad y Ambiente |
| **Operations** | Operaciones |
| **HR** | Recursos Humanos |
| **Finance** | Finanzas |
| **IT** | Tecnología |
| **Quality** | Calidad |
| **Environmental** | Ambiental |
| **Security** | Seguridad física |
| **Ethics** | Ética y conducta |

### Estados de Políticas

| Estado | Descripción | Color |
|--------|-------------|-------|
| **Borrador** | En elaboración | Gris |
| **En Revisión** | Pendiente de aprobación | Amarillo |
| **Activa** | Vigente | Verde |
| **Reemplazada** | Sustituida por nueva versión | Azul |
| **Archivada** | Ya no aplica | Gris |

### Tipos de Permisos

| Tipo | Descripción |
|------|-------------|
| **Ambiental** | Permisos ambientales |
| **Operación** | Licencias de operación |
| **Construcción** | Permisos de construcción |
| **Transporte** | Permisos de transporte |
| **Importación** | Licencias de importación |

### Tipos de Auditoría

| Tipo | Descripción |
|------|-------------|
| **Interna** | Realizada por personal interno |
| **Externa** | Realizada por terceros |
| **Certificación** | Para obtener/mantener certificación |
| **Regulatoria** | Por entes gubernamentales |

## Relación con Otros Módulos

El módulo de Compliance se conecta con:

- **HSE**: Cumplimiento de normas de seguridad y ambiente.

- **Documentos**: Políticas y permisos se almacenan como documentos.

- **Calidad**: Auditorías de sistemas de gestión de calidad.

- **Contratos**: Cumplimiento de obligaciones contractuales.

- **Producción**: Reportes de producción a entes reguladores.
