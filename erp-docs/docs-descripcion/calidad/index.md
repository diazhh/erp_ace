# ✅ Control de Calidad

## ¿Qué es este módulo?

El módulo de **Control de Calidad** permite gestionar los procesos de aseguramiento de calidad: inspecciones, no conformidades, acciones correctivas y certificaciones. Es fundamental para garantizar que los productos y servicios cumplan con los estándares establecidos.

Piense en este módulo como su "sistema de gestión de calidad digital": registra inspecciones, detecta problemas, implementa correcciones y asegura la mejora continua.

## ¿Para quién es útil?

- **Personal de Calidad**: Para realizar inspecciones y gestionar NC
- **Supervisores de Producción**: Para verificar calidad de productos
- **Gerencia**: Para monitorear indicadores de calidad
- **Auditoría**: Para verificar cumplimiento de estándares

## ¿Qué puedo hacer aquí?

### Inspecciones de Calidad
- **Programar inspecciones** de recepción, proceso y final
- **Registrar resultados** (aprobada, fallida, condicional)
- **Documentar hallazgos**
- **Generar no conformidades** automáticamente

### No Conformidades (NC)
- **Registrar NC** detectadas
- **Clasificar por tipo** (menor, mayor, crítica)
- **Analizar causas** raíz
- **Definir acciones** correctivas
- **Dar seguimiento** hasta el cierre

### Acciones Correctivas
- **Crear acciones** para resolver NC
- **Asignar responsables**
- **Establecer fechas** límite
- **Verificar efectividad**

### Dashboard
- **Indicadores de calidad**: NC abiertas, inspecciones pendientes
- **Tendencias**: Gráficos de NC por período
- **Alertas**: NC vencidas, acciones pendientes

## Conceptos Importantes

### Tipos de Inspección

| Tipo | Descripción |
|------|-------------|
| **Recepción** | Al recibir materiales |
| **En Proceso** | Durante la producción |
| **Final** | Producto terminado |
| **Dimensional** | Mediciones físicas |
| **Visual** | Inspección visual |
| **Funcional** | Pruebas de funcionamiento |
| **Destructiva** | Pruebas que dañan la muestra |
| **No Destructiva** | Pruebas sin daño |

### Resultados de Inspección

| Resultado | Descripción | Color |
|-----------|-------------|-------|
| **Pendiente** | Sin evaluar | Gris |
| **Aprobada** | Cumple especificaciones | Verde |
| **Fallida** | No cumple | Rojo |
| **Condicional** | Cumple con observaciones | Amarillo |

### Tipos de No Conformidad

| Tipo | Descripción | Color |
|------|-------------|-------|
| **Menor** | Desviación leve, no afecta función | Verde |
| **Mayor** | Desviación significativa | Amarillo |
| **Crítica** | Afecta seguridad o función principal | Rojo |

### Categorías de NC

| Categoría | Descripción |
|-----------|-------------|
| **Material** | Problema con materias primas |
| **Mano de Obra** | Error humano |
| **Documentación** | Falta o error en documentos |
| **Proceso** | Falla en el proceso |
| **Equipo** | Problema con maquinaria |
| **Diseño** | Error de diseño |

### Estados de NC

| Estado | Descripción |
|--------|-------------|
| **Abierta** | Recién detectada |
| **En Análisis** | Investigando causa raíz |
| **Acción Pendiente** | Esperando definir acciones |
| **En Progreso** | Ejecutando acciones |
| **En Verificación** | Verificando efectividad |
| **Cerrada** | Resuelta completamente |

## Relación con Otros Módulos

El módulo de Calidad se conecta con:

- **Proyectos**: Las inspecciones y NC se asocian a proyectos.

- **Inventario**: Inspecciones de recepción de materiales.

- **Procura**: Verificación de calidad de compras.

- **HSE**: Las NC pueden generar incidentes de seguridad.

- **Documentos**: Procedimientos y registros de calidad.
