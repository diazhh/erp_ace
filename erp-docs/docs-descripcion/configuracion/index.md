# ⚙️ Configuración del Sistema

## ¿Qué es este módulo?

El módulo de **Configuración** permite a cada usuario personalizar su experiencia en el sistema. Incluye configuración de notificaciones por WhatsApp y Email, preferencias de tema (claro/oscuro) y notificaciones push del navegador.

Piense en este módulo como su "panel de preferencias personal": aquí configura cómo quiere recibir notificaciones y cómo se ve el sistema para usted.

## ¿Para quién es útil?

- **Todos los usuarios**: Para personalizar su experiencia
- **Administradores**: Para configurar integraciones del sistema

## ¿Qué puedo hacer aquí?

### Configuración de WhatsApp
- **Registrar número** de teléfono
- **Verificar** con código SMS
- **Activar/desactivar** notificaciones

### Configuración de Email
- **Registrar correo** electrónico
- **Verificar** con código
- **Activar/desactivar** notificaciones

### Tema Visual
- **Modo claro**: Fondo blanco
- **Modo oscuro**: Fondo oscuro

### Notificaciones Push
- **Activar** notificaciones del navegador
- **Probar** notificaciones

## Conceptos Importantes

### Verificación de WhatsApp

Para recibir notificaciones por WhatsApp:
1. Ingrese su número de teléfono
2. Seleccione el código de país
3. Solicite verificación
4. Ingrese el código recibido por SMS
5. ¡Listo! Recibirá notificaciones

### Verificación de Email

Para recibir notificaciones por correo:
1. Ingrese su dirección de email
2. Solicite verificación
3. Revise su bandeja de entrada
4. Ingrese el código recibido
5. ¡Listo! Recibirá notificaciones

### Modos de Tema

| Modo | Descripción |
|------|-------------|
| **Claro** | Fondo blanco, texto oscuro |
| **Oscuro** | Fondo oscuro, texto claro |

### Notificaciones Push

Las notificaciones push aparecen en su navegador/escritorio incluso cuando no está en el sistema. Requiere permiso del navegador.

## Configuración de Administrador

Los administradores tienen acceso adicional a:

### Configuración de WhatsApp (Sistema)
- Configurar la conexión de WhatsApp Business
- Ver estado de la conexión
- Gestionar sesión

### Configuración de Email (Sistema)
- Configurar servidor SMTP
- Plantillas de correo
- Configuración de envío

## Relación con Otros Módulos

El módulo de Configuración afecta a:

- **Notificaciones**: Determina cómo recibe alertas.

- **Todos los módulos**: El tema visual aplica a todo el sistema.
